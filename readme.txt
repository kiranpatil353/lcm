== Description ==
Location Click Map plugin is useful to display marker on clicking of location, use shortcode [location_map]., With fully customized icons and settings. Users can hide filter dropdown using [location_map hide_locations=1].


= A Few Feature Highlights =
* Display single and multiple Location Maps using shortcode  
* Click on location marker highlight.
* Custom icons
* Can customize infowindow by applying style to custom-location-info class.
* Free to use and customize.
* Different map types like Roadmap/Sattelite/Terrain/Hybrid.

== Installation ==

This section describes how to install the plugin and get it working.

In WordPress:

    Go to Plugins > Add New > search for :: Location Click Map ::
    Press Install Now for the Show notice or message on admin area plugin
    Press Activate Plugin
	then assign tags to categories available in your website

To install manually instead:

    Upload the Location-Click-Map  directory to the /wp-content/plugins/ directory
    Activate the plugin through the Plugins menu in WordPress
	then assign tags to categories available in your website
